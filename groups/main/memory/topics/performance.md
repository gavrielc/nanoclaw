# Performance Reviews

Historical record of weekly agent performance reviews. Newest entries at the top.

## Scoring Guide

| Dimension | Weight | What it measures |
|-----------|--------|-----------------|
| Quality | 30% | Rework rate, first-pass approval rate |
| Speed | 25% | Cycle time (DOING→REVIEW), throughput |
| Proactivity | 15% | Early blocker detection, memory contributions |
| Adherence | 20% | Governance compliance, sacred file usage |
| Cost-Benefit | 10% | Ext API efficiency, resource usage |

## Level Ladder

Observer → Advisor → Operator → Autonomous

- Promotion: 2 consecutive qualifying reviews (overall ≥4.0, trend ↑)
- Demotion: immediate on overall <2.0
- Max one level change per review
